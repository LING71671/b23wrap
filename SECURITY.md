# 安全说明

## 报告问题

若发现本仓库代码本身的安全缺陷（例如本地服务的路径穿越、意外绑定公网等），请通过 GitHub Security Advisory 或私密渠道联系维护者，**不要**在公开 Issue 中贴完整利用细节。

## 范围

| 属于本项目 | 不属于本项目 |
|---|---|
| b23wrap 本地服务与脚本缺陷 | B 站官方产品漏洞赏金 |
| 文档误导导致的误用说明 | 第三方短链生成网站 |

## 设计提示

- 默认监听 `127.0.0.1`，请勿在未加固时把服务暴露到公网
- 生成的 `b23.tv` 为真实短链，分发前请自行评估合规风险
